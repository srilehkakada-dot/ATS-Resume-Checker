# ATS Resume Checker

🔍 An AI-powered ATS (Applicant Tracking System) Resume Checker built using React, TypeScript, and Vite.

## 🚀 Live Demo :- https://sensational-caramel-dca04c.netlify.app

## 📌 Features

- ATS score calculation
- Resume analysis and feedback
- Keyword optimization suggestions
- Modern and responsive UI
- Fast performance with Vite
- Interactive user experience

## 🛠️ Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS

## 📂 Installation

```bash
git clone https://github.com/srilehkakada-dot/ATS-Resume-Checker.git
cd ATS-Resume-Checker
npm install
npm run dev
